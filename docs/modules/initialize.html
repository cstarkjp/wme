initialize.py¶

Config to run wm_erosion in IPython.

Sets up IPython environment if we’re running wm_erosion in a Jupyter notebook or Jupyter QtConsole.

  • prepares Matplotlib to display inline and at a ‘retina’ resolution – if this is not available, a benign error report is made and progress continues
  • enables automatic reloading of wm_erosion (in case the code has been modded) when a notebook is re-run in-situ

Requires matplotlib and IPython.

Uses IPython extensions autoreload.

The autoreload extension forces the wm_erosion package to be reloaded on restart. This makes code modding and subsequent rerunning of a notebook smooth and seamless. It is not needed for normal operation, and if unavailable processing continues regardless.
