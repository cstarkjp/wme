Prerequisites¶

wm_erosion is a Python 3.x package primarily intended to be invoked from a Jupyter notebook. Testing has been with Python 3.6.

Package dependencies¶

  • python \(\geq\) 3.6
  • matplotlib \(\geq\) 3.0.2
  • numpy \(\geq\) 1.15.0
  • scipy \(\geq\) 1.1.0
  • sympy \(\geq\) 1.3
  • pandas \(\geq\) 0.23.4
