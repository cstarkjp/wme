Installation¶

First, Git clone the repository:

git clone https://github.com/cstarkjp/WeatheringMediatedErosion.git

Then append the location of the python package to your PYTHONPATH environmental variable. For Linux/MacOSX users, mod your .bashrc something like this:

export WMEHOME="${HOME}/<path_to_clone>/WeatheringMediatedErosion"
export PYTHONPATH="${PYTHONPATH}:${WMEHOME}/python"

That’s it.

If Python fails to import the wm_erosion package, i.e. when doing this:

import wm_erosion as wme

it means the PYTHONPATH is not set correctly in the shell from which Python was invoked. Check the environment variable against the full path to the python/ parent folder of the modules.
